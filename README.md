## ⚡️ Requirements

- stylelint >= **^15.5.0**

## 📦 Installation

Run this command to install stylelint and the plugin.

```bash
pnpm i -D stylelint-config-coder-dev
```

## 🚀 Usage

To use this plugin put in your stylelint config file this lines of codes.

```json
{
  "extends": "stylelint-config-coder-dev"
}
```
